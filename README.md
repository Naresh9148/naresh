# 🏦 ATM Banking Web Application

## 📌 Overview

This is a web-based ATM banking system built using Python, Flask, and SQLite.
The application allows users to securely log in and perform basic banking operations through a modern dark-themed dashboard.

---

## 🚀 Features

* User authentication (Username & PIN)
* Check account balance
* Deposit money
* Withdraw money
* Transfer funds between users
* Persistent data storage using SQLite
* Professional dark-themed UI dashboard

---

## 🧠 Tech Stack

* Python
* Flask
* SQLite
* HTML & CSS

---

## ⚙️ Installation & Run

### 1. Clone repository

```bash
git clone https://github.com/your-username/atm-project.git
cd atm-project
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run application

```bash
python app.py
```

### 4. Open in browser

http://127.0.0.1:5000
